# projeto_farmacia
Criação de uma API com Spring framework. A API cadastra, deleta, altera e lista medicamentos e administradores. Esse projeto posteriormente será um sistema de controle de medicamentos para empresas farmacêuticas. No desenvolvimento foram utilizados todas as dependências mais comuns do Spring, como JPA criando o banco com Hibernate e Spring Web. Foi utilizado também o padrão de projeto MVC, model, view e controller. O banco de dados foi o MySQL.
